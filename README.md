# Как подключить Bluetooth-наушники к ноутбуку или ПК на ОС Linux

1. включить Bluetooth на ноутбуке:
    * открыть приложение Менеджер Bluetooth
    * переключить тумблер в положение Включить

2. включить наушники:
   * открыть кейс от наушников, чтобы загорелся зеленый индикатор
   * удерживать кнопку питания на кейсе от наушников в течение 3 секунд
   * дождаться мигания белого индикатора — наушники находятся в режиме сопряжения
3. осуществить поиск устройств для сопряжения:
   * нажать Поиск
   * найти наушники в списке устройств для подключения и выбрать их двойным щелчком мыши
